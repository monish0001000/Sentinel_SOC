Abstract

Project: Sentinel SOC - Level 8 Enterprise
Domain: Cybersecurity / Security Operations Center (SOC)
Version: 1.0.0

Sentinel SOC is a unified, real-time Security Operations Center architecture designed to demonstrate advanced hybrid defense capabilities. It integrates a decentralized agent network with a centralized, containerized command core to provide comprehensive monitoring, detection, and response.

The system features a **microservices architecture** managed via Docker, comprising a **C2 Core** (Command & Control) built on FastAPI and WebSocket for real-time communication, and a **Next.js/React-based Dashboard** for visualization. Key components include a **Blockchain-secured SIEM** (Security Information & Event Management) system that ensures log integrity through cryptographic hashing, and a **SOAR Engine** (Security Orchestration, Automation & Response) that executes automated playbooks for threats like ransomware and intrusion attempts.

Advanced intelligence is provided by an **AI Analyst** leveraging Large Language Models (Ollama) for log interpretation and threat scoring, alongside a **Predictive Engine** based on the MITRE ATT&CK framework to forecast attack paths. The system also supports a standalone **Windows Agent** for endpoint monitoring, file integrity checking, and local firewall management.

This project delivers a rapidly deployable, self-healing, and adaptive security infrastructure suitable for enterprise-grade simulation and research.
