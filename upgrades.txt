================================================================================
MIGRATION FROM v1.0 TO v2.0
================================================================================

If upgrading from existing v1.0 installation:

1. Backup existing databases:
   - sentinel_siem.db
   - incident.db

2. Install new dependencies:
   pip install -r requirements.txt

3. Run migration (if using old databases):
   python migrate_v1_to_v2.py

4. Install npcap driver:
   choco install npcap

5. Restart services:
   python c2_core/main.py

6. Verify all collectors are running:
   - System logs collector
   - Network monitor
   - EDR behavior engine
   - FIM monitor
   - Threat intelligence engine
